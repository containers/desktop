# Installing the `lima` CLI

#### Procedure

- The `limactl` executable is installed.
  See [Installing Lima](https://lima-vm.io/docs/installation/).

  ```shell-session
  $ brew install lima
  ```

#### Verification

1. You can run the `lima` CLI:

   ```shell-session
   $ lima
   ```
