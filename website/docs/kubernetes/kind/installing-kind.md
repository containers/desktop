# Installing the `kind` CLI

#### Procedure

* In the status bar, click on the **Kind** button and follow the prompts.

#### Verification

1. The status bar doesn't display **Kind**.
1. **Settings > Resources** contain a **Kind** tile.
1. You can run the `kind` CLI:
   ```shell-session
   $ kind get clusters
   ```
